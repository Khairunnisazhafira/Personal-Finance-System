# Langkah-langkah
1. Download file
   clone project dengan
   ```
   https://github.com/Khairunnisazhafira/Personal-Finance-System/edit/main/README.md
   ```
2. Buat databe 
3. Ubah file .env.example menjadi .env
4. ```
   $ php artisan key:generate
   ```
5. ```
   $ php artisan migrate
   ```
6. ```
   $ php artisan db:seed
   ```
7. ```
   $ php artisan serve
   ```
8. copy paste link yang ada
   ```
    http://127.0.0.1:8000
   ```
